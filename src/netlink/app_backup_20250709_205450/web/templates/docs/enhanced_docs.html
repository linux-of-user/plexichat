What is NetLink?

NetLink is a government-level secure communication platform designed for organizations requiring the highest levels of security, reliability, and control. Built with enterprise-grade features and military-level encryption, NetLink provides a comprehensive solution for secure messaging, file sharing, and team collaboration.

Key Features

  • Government-Level Security: Advanced encryption, 2FA, and comprehensive security measures
  • Unified Administration: Single admin interface accessible via web and desktop applications
  • Advanced Backup System: Distributed shard-based backup with automatic redundancy
  • Modular Architecture: Plugin system with comprehensive management tools
  • Real-time Communication: Secure messaging with emoji support and file sharing
  • Comprehensive Monitoring: System health monitoring and detailed logging

  • Database Encryption: AES-256 encryption for all stored data

Getting Started

To get started with NetLink:

  1. Complete the installation process (see Installation Guide)
  2. Access the admin panel using the auto-generated credentials
  3. Change the default password and enable 2FA
  4. Configure your security settings and backup system
  5. Create user accounts and configure permissions

Authentication API

NetLink uses government-level authentication with session-based security and optional 2FA.

Login Endpoint

POST /api/auth/login

Authenticate user and create secure session

Request Body:

{
  "username": "admin",
  "password": "your_secure_password",
  "totp_code": "123456"  // Optional 2FA code
}

Response:

{
  "success": true,
  "message": "Login successful",
  "session_token": "secure_session_token",
  "must_change_password": false,
  "requires_2fa": false
}

Password Change

POST /api/auth/change-password

Change user password with validation

Request Body:

{
  "current_password": "current_password",
  "new_password": "new_secure_password"
}
