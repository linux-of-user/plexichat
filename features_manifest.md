# PlexiChat Features Manifest

## Core Features

- [ ] **User Management**: Registration, login, profile management, role-based access control.
- [ ] **Messaging**: Text, emoji, file attachments, markdown support, message editing/deletion.
- [ ] **Real-Time Updates**: WebSocket-based event system for instant message delivery and status updates.
- [ ] **Channels & Guilds**: Organized communication spaces with granular permissions.

## Advanced Features

- [ ] **Voice & Video**: High-quality, low-latency calls using WebRTC.
- [ ] **Screen Sharing**: Ability to share screens during calls.
- [ ] **End-to-End Encryption**: Optional E2EE for private conversations.
- [ ] **Post-Quantum Cryptography**: Kyber/Dilithium integration for key exchange and signing.

## System Features

- [ ] **Unified Configuration**: Single `config.yaml` for all settings.
- [ ] **Plugin System**: Dynamic loading of Python-based plugins.
- [ ] **Log Rotation**: Automatic rotation and compression of log files.
- [ ] **Database Abstraction**: Support for SQLite and PostgreSQL.

## Management Features

- [ ] **CLI**: Advanced command-line tool for server operations.
- [ ] **WebUI**: Admin dashboard for configuration and monitoring.
- [ ] **Metrics & Monitoring**: Built-in performance tracking and health checks.
